# 俄罗斯方块背景音乐

这里可以放置俄罗斯方块游戏的背景音乐文件。

## 经典音乐

经典俄罗斯方块游戏使用的背景音乐是《Korobeiniki》（Коробейники），这是一首俄罗斯民歌。

## 如何添加音乐文件

1. 将音乐文件放置在此目录下，命名为 `bgmusic.mp3` 或 `bgmusic.ogg`
2. 支持的格式：MP3、OGG、WAV
3. 建议使用循环播放的版本，音乐会自动循环播放
